## Global shortcuts

These work from **any application** — NIOM intercepts them at the OS level.

| Shortcut | Action |
|:---------|:-------|
| `⌥ Space` | Summon/dismiss NIOM overlay |

## In-app shortcuts

These work when the NIOM window is focused.

### Navigation

| Shortcut | Action |
|:---------|:-------|
| `⌘ N` | New thread |
| `⌘ K` | Quick command palette |
| `⌘ ,` | Open Settings |
| `⌘ 1` | Switch to Home view |
| `⌘ 2` | Switch to Chat view |

### Chat

| Shortcut | Action |
|:---------|:-------|
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `Escape` | Cancel streaming response |
| `⌘ Shift + V` | Toggle voice input |

### Tasks

| Shortcut | Action |
|:---------|:-------|
| `⌘ T` | Toggle task mode in prompt box |

<Tip>
  The global hotkey (`⌥ Space`) is the most powerful shortcut. It summons NIOM from anywhere — type a quick question, get an answer, and press `⌥ Space` again to dismiss. No window switching required.
</Tip>

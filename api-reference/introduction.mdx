title: "API Reference"
description: "Complete HTTP API for the NIOM sidecar — agent, tasks, providers, MCP, and threads."
---

The NIOM sidecar exposes a REST + SSE API on `localhost:3001`. The React overlay communicates with it via HTTP, and you can use it directly for automation and integration.

## Base URL

```
http://localhost:3001
```

## Authentication

No authentication required — the sidecar only binds to `localhost`.

## Endpoints Overview

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/health` | Health check |
| `GET` | `/` | API info + active model |
| **Agent** | | |
| `POST` | `/run` | Run agent (SSE stream) |
| `POST` | `/run/approve` | Resume after tool approval (SSE stream) |
| `POST` | `/run/sync` | Run agent (JSON response, for testing) |
| **Tasks** | | |
| `GET` | `/tasks` | List all tasks |
| `POST` | `/tasks` | Create a task |
| `GET` | `/tasks/:id` | Get task details |
| `PUT` | `/tasks/:id` | Update task |
| `DELETE` | `/tasks/:id` | Delete task |
| `POST` | `/tasks/:id/start` | Start a planned task |
| `POST` | `/tasks/:id/pause` | Pause a running task |
| `POST` | `/tasks/:id/resume` | Resume a paused task |
| `POST` | `/tasks/:id/run` | Trigger immediate execution |
| `POST` | `/tasks/:id/approve` | Approve or reject a pending run |
| `GET` | `/tasks/:id/runs` | Execution history |
| `GET` | `/tasks/events` | Real-time progress (SSE) |
| **Providers** | | |
| `GET` | `/providers` | List providers |
| `GET` | `/models` | List available models |
| `POST` | `/providers/configure` | Switch model/provider |
| `POST` | `/providers/test` | Test connectivity |
| **Threads** | | |
| `GET` | `/threads` | List conversation threads |
| `GET` | `/threads/:id` | Get thread |
| `PUT` | `/threads/:id` | Save/update thread |
| `DELETE` | `/threads/:id` | Delete thread |
| `DELETE` | `/threads` | Clear all threads |
| **MCP** | | |
| `GET` | `/mcp/servers` | List connected MCP servers |
| `POST` | `/mcp/connect` | Connect to an MCP server |
| `DELETE` | `/mcp/:name` | Disconnect an MCP server |

NIOM is a desktop AI agent that runs entirely on your machine. It remembers what you tell it, understands your projects, picks the right tools for every request, and executes tasks in the background while you keep working.

It's not a chat wrapper. It's a system that gets smarter the more you use it.

<Note>
  NIOM is open source and free. All data stays on your machine. [View on GitHub →](https://github.com/niomstack/niom)
</Note>

## What makes NIOM different

<CardGroup cols={2}>
  <Card title="Persistent memory" icon="brain">
    Every conversation teaches NIOM something. It extracts facts, preferences, and patterns — then uses them in future conversations. "I prefer TypeScript strict mode" is remembered once, applied forever.
  </Card>
  <Card title="Intelligent Skill Routing" icon="sitemap">
    Every request flows through a Skill Tree — a graph of tools organized by domain. NIOM picks the right 4-8 tools for your task, not all 20+. Zero tokens used for routing.
  </Card>
  <Card title="Background tasks" icon="signal">
    Dispatch multi-step tasks that run autonomously: plan → execute → checkpoint → deliver. Pause, steer, or cancel anytime. Tasks survive app restarts.
  </Card>
  <Card title="Project awareness" icon="folder-tree">
    NIOM auto-detects your workspaces — tech stacks, conventions, project structure. It knows you're in a React project before you tell it.
  </Card>
  <Card title="Multi-provider" icon="arrows-split-up-and-left">
    Anthropic, OpenAI, Google, or Ollama. Switch models anytime. Use Claude for reasoning, GPT for vision, Llama for privacy — in the same app.
  </Card>
  <Card title="Summoned in a keystroke" icon="keyboard">
    Press `⌥ Space` from any app to summon NIOM. Ask a question, run a command, start a task — and dismiss. It lives in your system tray, always ready.
  </Card>

|:-----------|:-----------------|
| A **developer** | Read/write files, run shell commands, search codebases, manage Git workflows — all through conversation |
| A **researcher** | Search the web, crawl pages, synthesize findings, produce reports — as background tasks |
| A **power user** | Automate repetitive workflows, manage projects, organize information across domains |
| **Privacy-focused** | Run fully offline with Ollama — zero API calls, zero data leaves your machine |

## How it works

```
You type → Skill Tree routes → Right tools selected → Agent executes → Memory updated → Results streamed
```

NIOM runs as an Electron desktop app with two processes:

1. **Main process** — Agent engine, tool execution, memory, Skill Tree routing, task orchestration
2. **Renderer** — React UI with chat, threads, settings, task panel, and a context graph

Press `⌥ Space` to summon the overlay from anywhere. It works on macOS, Windows, and Linux.

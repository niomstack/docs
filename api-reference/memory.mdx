---
title: "Memory API"
description: "View, add, and manage NIOM's long-term memory — facts, preferences, and patterns."
---

## Endpoints

| Method | Path | Description |
|:-------|:-----|:------------|
| `GET` | `/memory/brain` | View all brain data |
| `POST` | `/memory/brain/fact` | Add a fact |
| `DELETE` | `/memory/brain/fact` | Remove a fact |
| `POST` | `/memory/brain/pref` | Set a preference |
| `DELETE` | `/memory/brain` | Clear all brain data |

---

## View Brain

<ParamField path="GET /memory/brain" />

Returns all stored brain data and a preview of the context that gets injected into the system prompt.

**Response:**

```json
{
  "brain": {
    "facts": ["User's name is Arka", "Prefers TypeScript"],
    "preferences": { "editor": "VS Code", "style": "concise" },
    "patterns": [],
    "updatedAt": 1709384400000
  },
  "contextPreview": "Known facts about this user:\n- User's name is Arka\n- Prefers TypeScript\n\nUser preferences:\n- editor: VS Code\n- style: concise"
}
```

---

## Add Fact

<ParamField path="POST /memory/brain/fact" />

Teaches NIOM a new fact about you.

**Request body:**

```json
{
  "fact": "I work on the NIOM project"
}
```

**Response:**

```json
{
  "status": "learned",
  "brain": { "facts": [...], "preferences": {...}, "patterns": [...], "updatedAt": 1709384400000 }
}
```

---

## Remove Fact

<ParamField path="DELETE /memory/brain/fact" />

Removes a specific fact from memory.

**Request body:**

```json
{
  "fact": "I work on the NIOM project"
}
```

**Response (200):**

```json
{
  "status": "removed",
  "brain": { "facts": [...], "preferences": {...}, "patterns": [...], "updatedAt": 1709384400000 }
}
```

**Response (404):**

```json
{
  "error": "Fact not found"
}
```

---

## Set Preference

<ParamField path="POST /memory/brain/pref" />

Sets a key-value preference. Overwrites if the key already exists.

**Request body:**

```json
{
  "key": "language",
  "value": "TypeScript"
}
```

**Response:**

```json
{
  "status": "set",
  "brain": { "facts": [...], "preferences": {...}, "patterns": [...], "updatedAt": 1709384400000 }
}
```

---

## Clear Brain

<ParamField path="DELETE /memory/brain" />

Permanently deletes all brain data — facts, preferences, and patterns. This cannot be undone.

**Response:**

```json
{
  "status": "cleared"
}
```
